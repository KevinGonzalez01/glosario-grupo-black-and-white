 Software
Es el conjunto de programas y aplicaciones que hacen funcionar una computadora o dispositivo.
Ejemplos: sistemas operativos, aplicaciones móviles, programas de oficina.

2. Hardware
## Terminos de Hardware
### `CPU` — Unidad Central de Procesamiento
Es el cerebro del computador. Procesa todas las instrucciones
que le damos al sistema. Sin el, ninguna operacion es posible.
### `RAM` — Memoria de Acceso Aleatorio
Es la memoria temporal del computador. Guarda lo que estas
usando en este momento. Al apagar el computador, se borra.
### `SSD` — Disco de Estado Solido
Es donde se guarda la informacion de forma permanente.
Es mas rapido que un disco duro tradicional (HDD).

3. Sistema Operativo (SO)
Es el software principal que controla el hardware y permite usar la computadora.
Ejemplos:
*Microsoft Windows
*macOS
*Linux
*Android
