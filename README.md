 Software
Es el conjunto de programas y aplicaciones que hacen funcionar una computadora o dispositivo.
Ejemplos: sistemas operativos, aplicaciones móviles, programas de oficina.

2. Hardware
Son las partes físicas del computador.
Ejemplos: teclado, monitor, disco duro, memoria RAM.

3. Sistema Operativo (SO)
Es el software principal que controla el hardware y permite usar la computadora.
Ejemplos:
*Microsoft Windows
*macOS
*Linux
*Android

4. Aplicación (App)
Es un programa creado para realizar una tarea específica para el usuario.
Ejemplos: navegadores, juegos, redes sociales.

5. Programación
Es el proceso de escribir instrucciones (código) para que una computadora realice tareas.

6. Código fuente
Es el conjunto de instrucciones escritas por los programadores en un lenguaje de programación.

7. Base de datos
Es un sistema que almacena información organizada para poder consultarla fácilmente.

8. Seguridad informática
Son técnicas para proteger sistemas, redes y datos contra ataques o accesos no autorizados.

9. Computación en la nube (Cloud)
Es el uso de servidores en internet para guardar datos o ejecutar software, en lugar de hacerlo solo en tu computadora.

10. Actualización (Update)
Es una nueva versión de un software que corrige errores, mejora funciones o añade características.
